Installing JuiceboxBuilder-Pro

JuiceboxBuilder-Pro is the easiest way to create Juicebox-Pro galleries on your desktop. Follow these steps to install:

  1. If you don't have it already, download and install Adobe AIR.
  2. Double-click the 'JuiceboxBuilder-Pro.air' file to install the application.
  3. Run the application to build your first gallery.

View the Getting Started Guide.

Installing Adobe Lightroom Plugin

To create Juicebox-Pro galleries with Adobe Lightroom, install the Juicebox-Pro Lightroom plugin:

  1. Double click the 'juicebox_pro.lrwebengine' file inside the 'adobe-lightroom-plugin' folder to install the plugin.
  2. Restart Lightroom.
  3. In Lightroom, go to the 'Web' panel. Select 'Juicebox-Pro' from the 'Layout Style' Menu.

View the Lightroom Plugin Support Page.
